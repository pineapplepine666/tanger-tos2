SB BOT 隱私權政策

致力於數據透明化與用戶隱私保護的專業標準

核心原則與承諾

我們堅信隱私是一項基本人權，不僅是法律要求，幕後更是我們服務的基石。在處理您的數據時，我們維持最高標準的誠信與透明度。

政策適用範圍 (Scope)

服務對象

本政策適用於所有將 SB BOT 加入其 Discord 伺服器的管理員，以及在該環境中與機器人互動的任何終端用戶。

涵蓋技術

包含 Discord 斜線指令互動、伺服器元數據處理，以及任何透過我們的後端 API 傳輸的技術日誌。

數據收集三部曲

識別資訊

Discord 使用者 ID、伺服器 ID 與頻道 ID。這是維持運行的最小必要數據。

指令日誌

斜線指令及其參數紀錄，僅用於執行邏輯與修復程式崩潰 (Bugs)。

公開設定

您的 Discord 用戶名稱與頭像，以及伺服器的基本公開配置資訊。

數據用途與服務優化

  • 執行核心功能
    包括身分驗證、權限管理與系統響應。
  • 安全性維護
    監測惡意干擾、洗頻或任何違反 Discord 條款的行為。
  • 系統除錯
    透過錯誤日誌追蹤修復程序漏洞，提升穩定性。

您可以隨時行使您的權利：從查閱我們收集的資訊，到要求徹底手動刪除您的所有數據記錄。在 SB BOT 的世界裡，您才是數據的真正主人。

安全性防禦協議

加密存儲

所有核心資料庫皆採用業界標準加密保護。

權限隔離

僅授權開發人員在維護時可訪問必要數據。

定期抹除

暫時性日誌在任務完成後將自動銷毀。

未成年人保護

COPPA 合規聲明

SB BOT 並非為 13 歲以下兒童設計。我們不會在知情的情況下收集該族群的任何資訊。若您發現您的孩子使用了本服務，請隨時聯絡我們進行數據抹除。

兒童安全是我們不可逾越的紅線。

數據處理簡要流程

1. 數據收集

使用者發送指令時觸發最小數據點獲取。

2. 功能處理

數據由加密後端處理以回傳用戶請求。

3. 存儲日誌

必要的錯誤日誌在雲端暫存以供除錯。

4. 循環抹除

數據在清理週期後徹底從伺服器移除。
